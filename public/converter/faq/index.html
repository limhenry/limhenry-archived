  • What is Smart Converter?

    This Smart Converter is the combination of three single converter. We combined time zone converter, unit converter and currency converter into one, single program, you can convert it just a few clicks.

  • What can I do with Smart Converter?

    You can convert time zone, currency and units just in one, single program.

  • Can I run Smart Converter on my computer?

    If you are running Windows, Linux or Mac OS X, yes. This Smart Converter is a cross-platform program which support Windows, Linux and Mac OS X. You can download it from our download page or here.

  • How to install Smart Converter?

    For Windows:

    We are providing two way for you to run this program.


    Installer (Recommended)

    This is the easiest and the recommended way to install and to run it. Download the installer from our download page. Run the installer and follow the step provided in the installer. A shortcut for Smart Converter will appear when you done the installation. Click on it to run the program.


    Portable Version

    Download the file from our download page. You should see a “Portable Version” there, click it to download. After the download finished, open the file and unzip it. Double click the execute file with the name “Smart Converter.exe”.



    For Linux and Mac OS X:

    Installation is not required. Just download the file and you are done. Unzip the file and run it. You should see a file with the name “Smart Converter”. Double click to run it.
